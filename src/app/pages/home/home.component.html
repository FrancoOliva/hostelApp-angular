Antes de la pandemia trabajé en un hostel en donde usaban planillas para todo: registrar clientes, pagos, gastos, hasta tenían las habitaciones con las camas dibujadas. Esto generaba problemas cada tanto.

Durante mi curso intensivo en Comunidad IT, recordé este problema y desarrollé esta misma aplicación pero utilizando Framework7, que se utiliza para el desarrollo de aplicaciones híbridas.

En esta oportunidad utilicé: Angular 12, Typescript, PrimeNG, Firebase para mi backend y la autenticación de usuarios, Animate CSS para las animaciones y REST COUNTRIES para realizar peticiones HTTP (API de países).

Muchas gracias por tomarte un tiempo para conocer esta aplicación. Cualquier consulta, no dudes en contactarme.
